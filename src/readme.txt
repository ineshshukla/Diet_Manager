# Diet Manager Program

## Steps to Use the Program

1. **Setup**:
   - Ensure you have Rust installed on your system.
   - Navigate to the project directory.
   - Run `cargo build` to compile the program.

2. **Running the Program**:
   - Execute the program using `cargo run`.

3. **Features**:
   - **Add Basic Food**: Add a food item with a unique identifier, keywords, and calorie information.
   - **Add Composite Food**: Create a composite food by combining multiple basic foods.
   - **List Foods**: View all foods in the database.
   - **Search Foods**: Search for foods using keywords.
   - **View Daily Log**: Display logged food entries for the current date.
   - **Log Food**: Add a food entry to the daily log.
   - **Remove Log Entry**: Remove a specific food entry from the daily log.
   - **Edit Profile**: Update user profile details like age, weight, and activity level.
   - **Set Daily Target**: Set or remove a custom daily calorie target.
   - **View Daily Summary**: View a summary of calories consumed versus the target.

4. **Navigation**:
   - Use options to navigate between dates or undo the last action.

5. **Saving Data**:
   - Save all changes using the `Save` option before exiting.

6. **Exiting**:
   - Use the `Exit` option to close the program safely.
